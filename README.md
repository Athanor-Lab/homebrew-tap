# Athanor-Lab Homebrew tap

Homebrew formulae for [Hyperflux](https://github.com/Athanor-Lab/hyperflux), a lightweight CLI download accelerator (fork of Axel) with URL range/list expansion.

## Install

```sh
brew install Athanor-Lab/tap/hyperflux
```

The installed command is `flux`:

```sh
flux 'https://example.com/video[01-12].mp4'
```

Latest from `master`:

```sh
brew install --HEAD Athanor-Lab/tap/hyperflux
```
